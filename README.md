# 2KBattlegroundsTools
Set of moddig tools for WWE 2K Battlegrounds


## String Editor
Allows for customization of specific text strings in the game (Wrestlers, 'Fun Facts', Locations ec). This tool can also add strings to the game, for other modding purposes.
